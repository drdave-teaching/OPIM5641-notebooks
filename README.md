# OPIM 5641 — Business Decision Modeling · Notebooks

Student-facing Google Colab notebooks for **OPIM 5641**, companions to the
[course companion book](https://drdave-teaching.github.io/opim5641-textbook/).

Organized by **topic**, in course order — the folders *are* the syllabus:

| Folder | Topic | Competency |
|---|---|---|
| `1_EDA_Intro/` | Python & data fundamentals | programming · pandas · data |
| `2_MonteCarlo/` | Monte Carlo simulation & uncertainty | distributions · parametric/nonparametric MC · risk |
| `3_BruteForce/` | Optimization foundations & brute force | the 3 building blocks · enumeration |
| `4_Graphical/` | Linear programming — graphical method | feasible region · corners · edge cases |
| `5_Simplex/` | Linear programming — Simplex (by hand) | augmented form · pivoting · duality |
| `6_Pyomo_LP/` | LP in Pyomo | allocation · covering · blending · shadow prices |
| `7_Nonlinear/` | Nonlinear & portfolio | polynomials · curve-fit · regression · Markowitz |
| `8_Integer/` | Integer & logical programming | binary vars · linking constraints · project selection |
| `9_Networks/` | Network models | transport · shortest path · assignment · TSP |
| `Project/` | Final project scaffold | portfolio optimization end-to-end |
| `Assignments/` | Individual assignment templates | — |
| `_archive/` | Retired / off-topic / superseded (kept, not deleted) | — |

*Within each folder, `_blank` notebooks are the in-class (live) drivers; the worked/answer
versions are the pre-recorded (async) companions. The `pre-cleanup-2026-07-08` git tag preserves
the pre-reorg state, so nothing is ever lost.*
