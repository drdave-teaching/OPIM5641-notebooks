# OPIM 5641 - Studio Warm-Up Quizzes

Quick 5-question quizzes to **open each in-person studio**, the week after students watch the async videos. They "flex the knowledge" from the async and get everyone warmed up before we build live.

- ~5-10 minutes, low-stakes (great as a think-pair-share or a quick Kahoot).
- Each mixes a **concept check**, a **read-the-code** item, and a **why-it-matters** question.
- Answer key is at the bottom of each file - do them first!
